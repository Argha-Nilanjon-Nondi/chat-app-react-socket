
# Video Chat App

It is a message chat app .(realtime comunication)



## Features
- One to one user 
- Many to many users
- Create room
- Delete room
- Join room
- End to end encryption


## Built with
- HTML5
- CSS3
- Javascript
- React js
- Node js
- Socket.io
- CryptoJS

## Deployment

To strat the react server

```bash
  cd chat-frontend
  npm start
```
To start the socket server
```bash
   cd chat-backend
   npm run dev
```
